\doxysubsection{Descripción detallada}
Bit\+Set con \mbox{\hyperlink{classdomini_1_1BitSetL_ad1e391d27dc99ed80d23d77bb8d5ffa3}{length()}} específica. 

Bit\+Set con length distinta al Bit\+Set estandard. El length de java.\+util.\+Bit\+Set devuelve el numero de bits hasta el último 1. \mbox{\hyperlink{classdomini_1_1BitSetL}{Bit\+SetL}} devuelve el numero de bits del último bit modificado o el especificado al construirse el \mbox{\hyperlink{classdomini_1_1BitSetL}{Bit\+SetL}}.
