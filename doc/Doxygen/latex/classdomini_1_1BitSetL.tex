\begin{DoxyAuthor}{Autor}
***REMOVED***

Bit\+Set con length distinta al Bit\+Set estandard. El length de java.\+util.\+Bit\+Set devuelve el numero de bits hasta el último 1. \mbox{\hyperlink{classdomini_1_1BitSetL}{Bit\+SetL}} devuelve el numero de bits del último bit modificado o el especificado al construir-\/se el \mbox{\hyperlink{classdomini_1_1BitSetL}{Bit\+SetL}}.
